# RCC 2026

Site statique du Racing Club Cubzaguais avec Decap CMS.

## Administration

Une fois déployé sur Netlify :

1. Activer Netlify Identity.
2. Activer Git Gateway.
3. Inviter les administrateurs.
4. Ouvrir `/admin/`.

## Contenus éditables

- Matchs : `outputs/data/matches.json`
- Actualités : `outputs/data/news.json`
- Paramètres : `outputs/data/settings.json`
- Seniors : `outputs/data/senior.json`
- École de rugby : `outputs/data/academy.json`
- Pôle jeunes : `outputs/data/youth.json`
